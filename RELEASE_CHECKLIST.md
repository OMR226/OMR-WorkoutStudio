# Release Checklist

Before publishing a new OMR WorkoutStudio release:

- [ ] Build the desktop app package.
- [ ] Confirm users can run the app without installing Python.
- [ ] Confirm FFmpeg is included or documented.
- [ ] Generate a test video with YouTube-safe music.
- [ ] Generate a test video with no music.
- [ ] Generate one specific day, one full week, and all weeks.
- [ ] Verify clean illustration mode is the default.
- [ ] Verify custom/API exercises are saved for future selection.
- [ ] Verify the 30-second intro shows summary + first exercise instructions.
- [ ] Upload the built ZIP to GitHub Releases.
- [ ] Update `docs/config.js` if the release file name changes.
- [ ] Confirm the website download button works.
